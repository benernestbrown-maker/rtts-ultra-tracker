RTTS Ultra Tracker - GitHub Pages App

This is the proper free app-style route, with no Netlify/V0 credits.

One-time setup:
1. Go to github.com and create/sign in to a free account.
2. Create a new public repository called rtts-ultra-tracker.
3. Upload the files from this ZIP: index.html, manifest.webmanifest, sw.js and the icon PNG files.
4. In the repository, go to Settings > Pages.
5. Under 'Build and deployment', choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Save.
7. GitHub will give you a URL like:
   https://YOURUSERNAME.github.io/rtts-ultra-tracker/

On iPhone:
1. Open that GitHub Pages URL.
2. Tap Share.
3. Tap Add to Home Screen.
4. Name it RTTS Tracker.
5. Open it from your Home Screen.

Your ticks, notes and weights save locally on the iPhone/browser.
